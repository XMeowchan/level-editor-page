# 🎮 平台跳跃关卡编辑器

一个简单易用的平台跳跃游戏关卡编辑器，支持实时测试和关卡导出。

## ✨ 功能特性

- 🎨 **可视化编辑** - 拖拽绘制，网格对齐
- ▶️ **实时测试** - 一键切换测试模式，完整物理引擎
- 🎮 **游戏元素** - 平台、尖刺、金币、出生点、终点
- 💾 **保存/加载** - 支持本地存储和文件导出
- 📱 **响应式设计** - 自适应不同屏幕尺寸

## 🎯 使用方法

### 编辑模式
1. 选择工具（平台、尖刺、金币等）
2. 在画布上点击或拖拽绘制
3. 使用橡皮擦删除元素

### 测试模式
- **移动**: WASD 或 方向键
- **跳跃**: 空格键
- 碰到尖刺或掉出地图会重生
- 收集金币，到达终点完成关卡
